# Uptime Daemon
## Setup
### Using Slack for Notifications
To use slack for notifications when a system goes down or comes back up, create a **.env** file in the **Sources/upd/** folder. Create the slack webhook. Copy the webhook url and paste it with the variable SLACK_WEBHOOK_URL=*your_url*. Copy the webhook workflow ID and paste it with the variable WORKFLOW_ID=*your_workflow_id*
